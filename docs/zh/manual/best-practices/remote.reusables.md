<template #LarePass-桌面端>
1. 打开 LarePass，点击主界面左上角头像区域。

2. 在弹出的面板中打开专用网络开关。

   ![在 LarePass 桌面端启用专用网络](/images/zh/manual/use-cases/vpn-desktop.png#bordered){width=60%}
</template>

:::info 添加配置文件
在 iOS 或 macOS 系统中启用专用网络时，需要将 VPN 配置文件添加到系统中。按照提示完成设置即可。
:::
